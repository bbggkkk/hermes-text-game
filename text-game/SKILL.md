# Hermes Text Game Skill

에이전트가 **게임 마스터(GM)**가 되어, 사용자가 원하는 모든 장르와 테마의 텍스트 게임을 직접 진행합니다. 별도의 코드나 엔진 스크립트 실행 없이 오직 프롬프트 템플릿에 따라 동작합니다.

---

## 작동 방식

### 1. 프롬프트 로드
에이전트는 이 스킬이 트리거되면, 반드시 [prompt_template.md](file:///home/hana/sources/@hermes/hermes-text-game/text-game/prompt_template.md) 파일의 내용을 읽고 숙지합니다.

### 2. 게임 시작
- 에이전트는 사용자에게 어떤 장르와 테마의 게임을 하고 싶은지 묻습니다. (예: "어떤 장르의 게임을 해볼까요? 판타지, SF, 추리, 로맨스 등 원하시는 배경을 말씀해 주세요!")
- 사용자가 장르를 결정하면, `prompt_template.md`에 명시된 **출력 템플릿**에 맞춰 바로 오프닝 씬을 렌더링하고 게임을 시작합니다.

### 3. 게임 진행
- 에이전트는 사용자의 입력을 받아 게임 내 세계관의 규칙과 상태에 따라 합리적으로 결과를 판정합니다.
- 매 턴마다 **반드시** `prompt_template.md`의 출력 템플릿 양식을 준수하여 응답해야 합니다.
- 스크립트나 외부 툴을 호출하지 않고, 에이전트의 자체적인 컨텍스트 기억 능력을 활용하여 상태(상태바, 이벤트 로그 등)를 유지합니다.
